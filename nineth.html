These are some most common text formatting tags in HTML-

  1. Br tag
    • This tag is use for line break in the text.
  2. Strong tag
    • This tag is use to define the text with strong importance.
  3. B tag
    • This tag is used to make the text bold.
  4. Big tag
    • This tag is used to increse the size of the text.
  5. Small tag
    • This tag is used to decrese the size of the text.
